# Security Policy

The full security policy — supported versions, vulnerability reporting, and secure coding
guidelines — lives in **[docs/SECURITY.md](docs/SECURITY.md)**.

## Quick Reference

| Version | Supported |
|---------|-----------|
| 4.x (latest) | ✅ Active support |
| < 4.0 | ❌ Not supported |

To report a vulnerability, open a
[GitHub Private Security Advisory](https://github.com/RajwanYair/RegiLattice/security/advisories/new).
